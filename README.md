# football
5v5 足球机器人 AI 决策与运动控制系统开发
## 项目概述
本项目面向5人制仿真机器人足球赛事，基于Python搭建分层解耦式自主决策框架，无需人工干预即可完成全场感知、路径规划、运动控制与攻防博弈，实现机器人自主跑位、传球、拦截、射门全套竞技行为。系统通过运动预测模型补偿传感器与执行器延迟，搭配自适应战术逻辑，大幅提升团队对抗稳定性与赛场胜率。

## 技术栈
Python、增量式PID控制、运动学轨迹预测、多智能体博弈、Socket通信、模块化分层架构

## 核心功能
1. 四层解耦架构：通信层、全局配置层、底层运动控制层、上层战术决策层，模块独立易扩展；
2. PID精准运动控制：封装定点移动、转向、射门原子接口，实现毫米级定位，运动平滑无抖动；
3. 前瞻状态预测机制：预判未来多帧机器人与足球位姿，抵消系统延迟，提升路径追踪精度；
4. 全场景战术库：内置进攻绕球、定点抽射、双人配合、联防防守、点球/门球专属策略；
5. 自适应博弈逻辑：根据赛场战况自动切换战术，进攻失败自动轮换方案，具备基础自主优化能力；
6. 多角色分工协同：划分前锋、中场、后卫、守门员，自动保持阵型，完成攻防无缝转换。

## 仓库文件说明
- V5RPC.py：仿真环境通信模块，标准化数据交互接口
- GlobalVariable.py：全局参数、赛场坐标常量统一管理
- baseRobot.py：PID运动控制、轨迹预测底层逻辑
- v5strategy.py：核心决策与全套攻防战术代码
- AI足球机器人策略项目.pptx：项目架构、算法、实战成果汇报文档
